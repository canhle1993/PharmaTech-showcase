# Project Summary

## Project Name

PharmaTech

## Project Type

Team-based semester 3 academic eProject focused on pharmaceutical equipment business workflows.

## Business Goal

Build a centralized digital platform to:

- present pharmaceutical equipment products
- receive quote and purchase requests
- manage customer and order workflows
- support recruitment and candidate tracking
- provide admin analytics and operations tools

## Technical Goal

Deliver a full-stack web application with:

- modern frontend
- modular backend API
- document and media upload handling
- real-time communication
- payment flow
- analytics dashboards
- deployment automation

## Outcome

The resulting system demonstrates both feature breadth and practical delivery:

- customer-facing website
- admin-facing management system
- recruitment analytics
- CI/CD deployment
- cloud-hosted live demo
