# Admin Features

The admin side of PharmaTech is a management workspace for commercial operations, recruitment, content control, and customer support.

## Account Management

- View all user accounts
- Search by name, email, phone, role, and status
- Create new admin accounts
- View account details including education, job preferences, skills, and resume
- Edit profile information
- Lock or restrict user access when needed

## Product and Category Management

- Manage product list with photo, model, price, quantity, and category
- Create, update, soft-delete, and restore products
- Manage category photo, name, description, and linked products
- View product detail and technical specification blocks
- Export product data to Excel

## Stock and Inventory

- View in-stock and out-of-stock products
- Update stock quantity individually or in bulk
- Export stock data for inventory review
- Generate QR codes for product units
- Scan QR history for stock verification and audit workflows

## Orders and Returns

- Review order list and order history
- Approve or reject deposit proof
- Track order status from pending to paid in full
- Review OCR output from payment proof
- Create and manage return requests
- Mark returns completed and keep status history

## Recruitment

- Manage career postings
- Track candidate applications
- Review profile and resume information
- Use recruitment analytics dashboards and charts
- Support HR communication flow with candidate emails

## Content, Quote, and Support

- Manage About content, Contact information, Hotline, and homepage content
- Update banners and media assets
- Review quote requests and send replies
- View support chat history
- Access AI-assisted support tools
