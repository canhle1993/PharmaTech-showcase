# Recruitment Features

Recruitment is one of the strongest parts of PharmaTech because it is not only a simple job listing page, but a full application management workflow.

## Candidate Side

- Create account and login
- Fill in detailed profile information
- Set preferred area, work type, salary, field, skills, languages, and education
- Upload resume / CV
- Browse careers and apply directly
- Track personal application status

## Admin / HR Side

- Create and manage job postings
- Review submitted applications
- Filter by status, department, or candidate profile
- View resume and candidate details
- Use charts for:
  - total applications
  - pending / interview / passed / rejected
  - gender distribution
  - age range
  - top skills
  - application funnel

## Value

This module shows a stronger business-oriented project scope than a typical student e-commerce app because it combines:

- business presentation
- hiring workflow
- candidate data management
- dashboard analytics
